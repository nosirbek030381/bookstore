## Boshlash

Birinchi ishlatish shaxsiy noutbukingizda:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Ochish [http://localhost:3000](http://localhost:3000) bilan va sizning brovzeringizda.

Stack:

- NextJS
- TypeScript
- TailwindCSS
- Firebase (auth, db)
- SEO
- Formik
- Toast
- SSR
- Material UI
